# Ecommerce Wgold

Este é um **e-commerce** da Empresa Wgold feita como projeto do curso de GitGithub do _Matheus Battisti_

## Funcionalidade:

_Checktou, **Tela de produto**, Catálogo, Home com banner_

   3. CSS diferente para grupo de clientes.
